# NINE77 — Premium Streetwear Store

NINE77 is a next-generation luxury streetwear e-commerce site built with Next.js, React, TypeScript, and Tailwind CSS. It features immersive 3D product visuals, smooth scroll interactions, and a clean, mobile-first shopping experience.

## Features

- 🛍️ Shop, product detail, FAQ, and about pages
- 🎨 Interactive 3D hero and product showcase scenes (React Three Fiber / Three.js)
- 🌀 Smooth scrolling powered by Lenis
- ✨ Motion and animation with Framer Motion and GSAP
- 📱 Mobile-friendly navigation (bottom nav + mobile menu)
- 📈 Analytics via Vercel Analytics
- 🔍 SEO-ready with sitemap and robots configuration

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- [React](https://react.dev/) 19
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [React Three Fiber](https://docs.pmnd.rs/react-three-fiber) & [Three.js](https://threejs.org/)
- [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://gsap.com/)
- [Lenis](https://lenis.darkroom.engineering/) for smooth scrolling

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Available Scripts

| Script          | Description                          |
| --------------- | ------------------------------------- |
| `npm run dev`   | Start the development server          |
| `npm run build` | Build the app for production          |
| `npm run start` | Start the production server           |
| `npm run lint`  | Run ESLint                            |
